Compatibility: This package is compatible with iOS iOS 13.4 to 14.x.

An ad-free manga reader for iOS and iPadOS. Paperback is an app for iOS that lets you read locally hosted comics/books/manga that you have purchased digitally. It also provides a robust API to allow anyone to interact with the UI Package uses wget, python 3, appsync, and appinst to update and install the app. To update the app, simply reinstall this package.
